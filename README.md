# VIDEOGAME - iLERNTALE

Bienvenido al repositorio oficial de **iLERNTALE**, un videojuego desarrollado íntegramente por nuestro equipo para el 1er curso de DAW. Este proyecto combina mecánicas de RPG clásico con desafíos de habilidad para crear una experiencia de aprendizaje interactiva.

---

## Nuestra Evolución

Como equipo, hemos avanzado siguiendo una hoja de ruta centrada en la calidad del código y la escalabilidad. Estos son los pasos que hemos dado hasta la fecha:

### Fase 1: Cimientos y Motor de Juego
*   **Arquitectura Base:** Establecimiento de la estructura del proyecto Maven (`DESIGN.md`) siguiendo la logica del juego segun el `GDD.md`.
*   **Creacion de Recursos:** Creación de la carpeta `resources` para almacenar las imágenes de los personajes, enemigos...etc.
*   **Ventana Principal:** Implementación de `MainFrame` y `GamePanel` inicial.

## Estructura del Proyecto

*   [`ilerntale/src/main/java/equipoilernatale/main`](ilerntale/src/main/java/equipoilernatale/main): Punto de entrada al juego.
*   [`ilerntale/src/main/java/equipoilernatale/model`](ilerntale/src/main/java/equipoilernatale/model): Lógica de personajes, atributos y estados.
*   [`ilerntale/src/main/java/equipoilernatale/view`](ilerntale/src/main/java/equipoilernatale/view): Interfaz gráfica, paneles y tiles.
*   [`ilerntale/src/main/java/equipoilernatale/controller`](ilerntale/src/main/java/equipoilernatale/controller): Lógica de control.
